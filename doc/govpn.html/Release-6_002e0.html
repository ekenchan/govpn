Release 6.0

  • Argon2d is replaced with Balloon hashing. Found Argon2 libraries written on pure Go have various problems. Moreover Argon2i should be used instead, but it has some possible cryptographic defects. So it is replaced with much more simpler (and seems even cryptographically better) Balloon hashing.
